# Nexus ERP System - Standalone Version

## Overview
This is a complete, self-contained ERP (Enterprise Resource Planning) system extracted as three standalone files:
- **index.html** - Complete HTML structure
- **styles.css** - All styling and responsive design
- **script.js** - Complete JavaScript functionality

## ⚙️ Requirements
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - runs entirely in the browser
- No internet connection needed (offline-first)

## 🚀 How to Use

1. **Open the System**
   - Double-click `index.html` to open in your default browser
   - Or right-click → Open with → Select your browser

2. **First Time Setup**
   - Go to Settings (⚙️) to configure:
     - Company name
     - Phone number
     - Address
     - Invoice footer text

3. **Main Features**
   - 📊 Dashboard - Overview of sales, inventory, expenses, and profit
   - 📦 Inventory - Manage products and stock levels
   - 🛒 Sales - Record customer invoices
   - 💰 Purchases - Track supplier purchases
   - 💸 Expenses - Record business expenses
   - 🏦 Debts - Manage customer and supplier debts
   - 📈 Profits - View profit analysis
   - 👥 Directory - Customer and supplier contacts
   - 📋 Reports - Summary reports and analytics
   - ⚙️ Settings - System configuration and data management

## 📊 Dashboard
- View total sales, stock value, expenses, and net profit
- 7-day sales vs purchases chart
- 7-day profit vs expenses analysis
- Auto-updating statistics

## 💾 Data Storage
- All data is stored locally in your browser
- Data persists even after closing the browser
- Uses browser's LocalStorage technology

## 📥 Import/Export
- **Export Data**: Settings → Export Data (saves as JSON)
- **Import Data**: Settings → Import Data (restore from JSON file)
- **Clear Data**: Settings → Delete All Data (irreversible)

## 🐛 Troubleshooting

**Data not saving?**
- Check that browser allows LocalStorage
- Try a different browser
- Clear browser cache and try again

**Charts not showing?**
- Ensure JavaScript is enabled
- Try refreshing the page
- Check browser console for errors

**Mobile issues?**
- Sidebar collapses - use menu toggle (☰)
- Swipe to navigate between views
- Tap outside modal to close
